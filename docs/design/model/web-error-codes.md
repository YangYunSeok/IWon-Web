# Web Admin 전용 에러 코드 (확장 SSOT)

> 공통 에러 모델과 1차 코드 리스트는 `docs/design/model/error-codes.md` 를 따른다.
> 본 문서는 관리자 웹 콘솔에서 추가로 사용하는 에러 코드/UX 매핑 정보를 관리한다.

## 1. 에러 코드

| 코드 | 설명 | 권장 UX |
| --- | --- | --- |
| DISCREPANCY_DETECTED | DB 잔액과 온체인 총량이 일치하지 않음 | 대시보드 Alert 섹션을 Red 로 표시하고 불일치 상세 링크 제공 |
| INSUFFICIENT_RESERVE | 준비금 부족으로 승인 불가 | 승인 상세 팝업에서 경고 모달을 띄우고 실행 버튼 비활성화 |
| WALLET_NOT_FOUND | 지갑 미생성 대상 포함 | 지갑 생성 버튼 또는 "지갑 미생성자 보기" 토글 강조 |
| APPROVAL_ALREADY_PROCESSED | 승인 요청이 이미 처리됨 | 승인 상세 팝업을 닫고 최신 목록 재조회 |

## 2. 에러 모델 재사용 규칙

- `code/message/details` 구조는 공통 문서를 그대로 사용한다.
- UI 문서에서는 위 코드들을 사용자 메시지/가이드 문구로 매핑한다.
- 새로운 관리자 전용 에러가 추가되면 **반드시 본 문서 → API 문서 → UI 문서 순으로 업데이트** 한다.
